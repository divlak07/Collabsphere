**CollabSphere Imperial** is a premium Streamlit-based influencer intelligence dashboard designed to help brands and agencies discover, evaluate, and optimize influencer collaborations using data-driven insights. It automatically ingests influencer data from a CSV file using intent-based column detection and fail-safe fallbacks, cleans and normalizes numeric values, and presents executive dashboards with reach, budget, and engagement metrics, along with advanced visualizations like ROI scatter plots and radar charts. The platform includes a talent acquisition module for budget- and niche-based filtering and an AI Strategy Vault that allows users to search influencers through natural-language queries using keyword matching and engagement-based ranking, making it a robust, visually rich, and production-ready internal tool for influencer marketing analysis.
